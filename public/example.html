Installation Instructions

Add this script to your website's <head> tag:

<script
  src="https://api.addwatch.site/tracker.js"
  data-api-key="YOUR_API_KEY_HERE"
  async
></script>

Note: Replace YOUR_API_KEY_HERE with your actual API key from the dashboard.

Important: The script automatically detects the API endpoint from the script URL. No need to specify data-endpoint manually!

Tracked Events Summary

  • Pageview: Automatically tracked on page load
  • Tel Click: Tracks clicks on <a href="tel:..."> links
  • WhatsApp Click: Tracks clicks on WhatsApp links (wa.me)
  • CTA Click: Tracks clicks on elements with class .track-cta
  • Form Submit: Tracks all form submissions

Additional Data Collected:

  • Visitor Type: New vs Returning (localStorage-based)
  • Device: Desktop, Mobile, or Tablet
  • Traffic Source: Google, Facebook, Instagram, YouTube, Direct, or Other
  • Location: Country and City (IP-based, from backend)
